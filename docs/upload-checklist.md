# GitHub 作品集上传清单

## 建议上传到仓库

- `README.md`
- `docs/project-overview.md`
- `docs/personal-contribution.md`
- `docs/ai-usage.md`
- `assets/screenshots/` 中的精选截图

## 不建议直接提交到仓库

- 完整工程压缩包：`DreamCard（工程文件）.rar`
- 游戏安装包：`游戏安装包.rar`
- 大体积展示视频：`游戏展示.mp4`

这些文件体积较大，建议放到 GitHub Releases、网盘或比赛提交平台，并在 README 中放链接。

## 推荐截图

- 主菜单截图
- 六边形地图截图
- 战斗界面截图
- 卡牌与倍率链截图
- 怪物或 Boss 战截图
- 胜利/结算截图

## 发布前检查

- 确认团队允许公开展示截图、视频和文档。
- 确认仓库中不包含未授权完整工程、美术源文件或比赛敏感资料。
- README 中明确个人职责，避免把团队成果写成个人独立完成。
- 如果公开安装包，优先使用 Release 或网盘链接，不直接提交大文件。
