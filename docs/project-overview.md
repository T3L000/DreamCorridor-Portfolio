# 项目介绍

《梦境回廊》是一款回合制卡牌策略 Roguelike。玩家扮演一位从农村搬到城市的小孩，在梦中回到儿时山林，用彩虹般的回忆对抗现实压力在梦境中的投影。

## 设计关键词

- 颜色链：卡牌同时拥有颜色与费用，出牌顺序会影响战斗节奏。
- 换手倍率：连续打出同费用、异颜色卡牌时提升倍率，放大伤害、护盾和治疗。
- 四色体系：红色偏攻击，蓝色偏防御与治疗，黄色偏抽牌与资源周转，绿色偏成长与反伤。
- 回廊地图：12 列六边形地图形成从开局、中段扩展到终局收束的路线体验。

## 基本玩法

玩家在地图中选择怪物、精英、商店、宝箱、篝火、随机事件和 Boss 房间。进入战斗后，通过拖拽卡牌攻击敌人或释放效果，并利用换手链提高倍率。战斗胜利后获得金币与卡牌奖励，逐步优化牌组并挑战最终 Boss。

## 作品定位

这个项目重点展示了卡牌机制设计、地图路线体验、怪物机制设计、团队协作开发和比赛版本落地能力。
